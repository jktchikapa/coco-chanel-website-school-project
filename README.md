# Site Web — Coco Chanel

Site web réalisé dans le cadre d’un projet scolaire, dédié à **Coco Chanel** et à son impact sur la mode.

## À propos
Le site présente la vie, le parcours et l’héritage de Coco Chanel à travers une navigation simple et un design épuré.
